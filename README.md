 **Lenia — Standalone EXE Version**

This repository contains a **standalone Windows executable** of **Lenia**, a continuous cellular automaton inspired by artificial life, originally created by Bert Chan.  
This build requires **no Python installation**, **no dependencies**, and runs on **any Windows device** out of the box.

---

 ✨ Features

- **Portable EXE** — no installation required  
- **Runs on any Windows machine** (Windows 7/8/10/11)  
- **Based on the classic 2D Lenia implementation**  
- **Fast startup and smooth simulation**  
- **Fully self‑contained** (Python interpreter + libraries bundled inside)  
- **Compatible with all hardware** — laptops, desktops, tablets, etc.

🧬 About Lenia

Lenia is a continuous cellular automaton that generalizes Conway’s Game of Life into a smooth, continuous domain.  
It produces a wide variety of lifelike, self‑organizing patterns such as:

- Orbium  
- Scutum  
- Paramecium‑like structures  
- Complex multi‑cell organisms  

For more information about the original project:

- Original repository: https://github.com/Chakazul/Lenia  
- Research paper: https://arxiv.org/abs/1812.05433
